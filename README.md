# Calculadora de Troco - Doces

Projeto React + Tailwind CSS para calcular troco em vendas de doces.

Passos para rodar localmente:

1. Instale dependências:

```bash
npm install
```

2. Rode em modo de desenvolvimento:

```bash
npm run dev
```

Observações:
- Alterar o `Valor Unitário` e clicar em `Salvar` grava no `localStorage`.
- Use os botões + / - para ajustar `Quantidade`.
- Clique numa nota para ver o `Troco a Voltar`.
